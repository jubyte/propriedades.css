Propriedades do Display

A propriedade display define como os elementos HTML são exibidos e organizados na página.
Ela controla se os elementos aparecem em bloco, em linha, ou em layouts mais avançados como flex e grid.

Display: block

Ocupa a largura toda e quebra linha antes/depois.

Display: inline

Fica na mesma linha e não aceita width/height.

Display: inline-block

Fica na mesma linha, mas aceita width/height.

Display: flex

Cria um container flexível para alinhar itens em linha ou coluna.

Display: inline-flex

Igual ao flex, mas o container fica inline.

Display: inline flex

Funciona como o flex, mas o container se comporta como um elemento inline, ocupando apenas o espaço do conteúdo e podendo ficar na mesma linha que outros elementos.

Display: block flex

Elemento se comporta como block externamente e usa layout flex internamente.

Display: grid

Cria um layout em grade usando linhas e colunas.

Display: inline-grid

Elemento se comporta como inline externamente e usa layout grid internamente.

Display: inline grid

Elemento se comporta como inline externamente e usa layout grid internamente.

Display: block grid

Elemento se comporta como block externamente e usa layout grid internamente.

Display: flow-root

Semelhante ao block, mas cria um novo contexto de formatação. Bom para conter elementos com float e evitar que o pai colapse de altura.

Display: block flow

Semelhante ao block.

Display: block flow-root

Semelhante ao block, mas cria um novo contexto de formatação.

Display: inline flow

Elemento inline com layout interno no fluxo normal, o pai se comporta como inline, já os filhos como flow.

Display: inline flow-root

Elemento inline que cria um novo contexto de layout.

Display: table

Faz o elemento se comportar como uma tabela HTML.
